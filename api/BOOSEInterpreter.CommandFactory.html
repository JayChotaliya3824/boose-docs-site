Class CommandFactory

Namespace
BOOSEInterpreter
Assembly
BOOSEapp.dll

The CommandFactory class is implemented to manage the creation and retrieval of command objects. The Factory design pattern is utilized to instantiate commands based on string identifiers, ensuring a decoupled architecture. The Singleton pattern is applied to ensure a single instance of the factory manages command generation throughout the application lifecycle.
